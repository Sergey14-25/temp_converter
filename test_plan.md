# Тест-план приложения "Конвертер температур"

## 1. Тестирование интерфейса

### 1.1. Проверка запуска приложения
- **Шаги:** Запустить `python main.py`
- **Ожидаемый результат:** Открывается окно "Конвертер температур v1.0"
- **Статус:** ✅ Пройден

### 1.2. Проверка отображения элементов
- **Шаги:** Осмотреть интерфейс
- **Ожидаемый результат:** Видны все элементы: поле ввода °C, три цветных блока, таблица, кнопки
- **Статус:** ✅ Пройден

### 1.3. Проверка адаптивности
- **Шаги:** Изменить размер окна
- **Ожидаемый результат:** Все элементы корректно масштабируются
- **Статус:** ✅ Пройден

## 2. Тестирование конвертации температур

### 2.1. Базовая конвертация
- **Шаги:** Ввести 25°C
- **Ожидаемый результат:** 77.00°F, 298.15K
- **Статус:** ✅ Пройден

### 2.2. Отрицательная температура
- **Шаги:** Ввести -40°C
- **Ожидаемый результат:** -40.00°F, 233.15K
- **Статус:** ✅ Пройден

### 2.3. Абсолютный ноль
- **Шаги:** Ввести -273.15°C
- **Ожидаемый результат:** -459.67°F, 0.00K
- **Статус:** ✅ Пройден

### 2.4. Температура ниже абсолютного нуля
- **Шаги:** Попытаться ввести -300°C
- **Ожидаемый результат:** QDoubleSpinBox не позволяет ввести значение ниже -273.15
- **Статус:** ✅ Пройден

## 3. Тестирование базы данных

### 3.1. Сохранение записи
- **Шаги:** Ввести температуру, нажать "Сохранить в историю"
- **Ожидаемый результат:** Запись появляется в таблице
- **Статус:** ✅ Пройден

### 3.2. Удаление записи
- **Шаги:** Выделить запись, нажать "Удалить выбранную запись"
- **Ожидаемый результат:** Запись удаляется из таблицы и БД
- **Статус:** ✅ Пройден

### 3.3. Экспорт в CSV
- **Шаги:** Нажать "Экспорт в CSV", выбрать файл
- **Ожидаемый результат:** Файл создаётся с корректными данными
- **Статус:** ✅ Пройден

## 4. Тестирование загрузки изображений

### 4.1. Загрузка фото термометра
- **Шаги:** Нажать "Загрузить фото термометра", выбрать изображение
- **Ожидаемый результат:** Изображение отображается в правой панели
- **Статус:** ✅ Пройден

## 5. Тестирование жизненного цикла

### 5.1. Корректное закрытие
- **Шаги:** Нажать крестик окна
- **Ожидаемый результат:** Появляется диалог подтверждения, БД закрывается
- **Статус:** ✅ Пройден
