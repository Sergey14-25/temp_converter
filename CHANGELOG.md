# История изменений проекта "Конвертер температур"

## Версия 1.0 (2026)

### Что добавлено:
- Создана структура проекта с тремя файлами: main.py, ui_main.py, database.py
- Сделан интерфейс на PyQt5 с использованием менеджеров компоновки
- Настроена конвертация температур между шкалами Цельсия, Фаренгейта и Кельвина
- Добавлено сохранение истории конвертаций в базу данных SQLite
- Сделана валидация ввода через QDoubleSpinBox (нельзя ввести температуру ниже -273.15°C)
- Настроено логирование всех операций приложения
- Добавлена обработка исключений во всех методах
- Реализована загрузка изображений через библиотеку Pillow
- Добавлена кнопка удаления записей из истории
- Добавлена кнопка экспорта данных в CSV файл
- Написана документация: README.md, report.md, test_plan.md
- Добавлены скриншоты работы приложения
