# Конвертер температур (PyQt5 Practice)

Приложение для конвертации температур между шкалами Цельсия, Фаренгейта и Кельвина с сохранением истории конвертаций.

## Возможности

- Конвертация температур: °C ↔ °F ↔ K
- Формулы: F = C × 9/5 + 32, K = C + 273.15
- Сохранение истории конвертаций в SQLite
- Загрузка фото термометра через Pillow
- Экспорт данных в CSV
- Удаление отдельных записей из истории
- Очистка всей истории одной кнопкой
- Валидация ввода через QDoubleSpinBox
- Адаптивный интерфейс с QSS-стилизацией

## Запуск проекта

Запуск проекта

Создайте виртуальное окружение: python -m venv venv
Активируйте:
Windows: venv\Scripts\activate
macOS/Linux: source venv/bin/activate
Установите зависимости: pip install -r requirements.txt
Запустите: python main.py
Структура проекта

main.py — точка входа, настройка QApplication и логирования
ui_main.py — главное окно: вёрстка, сигналы/слоты, CRUD-обработчики
database.py — слой данных: SQLite, CRUD-операции

Технологии

Python 3, PyQt5, SQLite3, Pillow
