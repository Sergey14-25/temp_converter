# Отчёт по практике: Разработка приложения "Конвертер температур"

## 1. Введение

### 1.1. Цель работы
Разработать десктопное приложение для конвертации температур между шкалами Цельсия, Фаренгейта и Кельвина с использованием Python и PyQt5.

### 1.2. Задачи
- Реализовать интерфейс с использованием менеджеров компоновки
- Настроить работу с базой данных SQLite
- Интегрировать библиотеку Pillow для работы с изображениями
- Реализовать логирование и обработку исключений
- Создать систему контроля версий Git

## 2. Описание проекта

### 2.1. Функциональные требования
- Конвертация температур по формулам: F = C × 9/5 + 32, K = C + 273.15
- Сохранение истории конвертаций в SQLite
- Загрузка изображений через Pillow
- Экспорт данных в CSV
- Валидация ввода (температура ≥ -273.15°C)

### 2.2. Нефункциональные требования
- Использование только менеджеров компоновки (без setGeometry/move)
- Параметризованные SQL-запросы (защита от инъекций)
- Логирование всех операций
- Обработка исключений во всех методах

## 4. Реализация

### 4.1. Слой данных (database.py)
Класс `DatabaseManager` реализует CRUD операции:
- `init_db()` — создание таблицы
- `insert_record()` — добавление записей с валидацией
- `get_all()` — получение всех записей
- `update_record()` — обновление записей
- `delete_record()` — удаление записей
- `close()` — закрытие соединения

### 4.2. Слой представления (ui_main.py)
Класс `MainWindow` реализует:
- Интерфейс через менеджеры компоновки (QVBoxLayout, QHBoxLayout, QGridLayout)
- Сигналы/слоты через `.connect()`
- QTimer для неблокирующей работы
- QDoubleSpinBox для валидированного ввода
- QSS для стилизации

### 4.3. Точка входа (main.py)
Функция `main()` настраивает:
- QApplication
- Логирование
- Главное окно
- Цикл обработки событий
